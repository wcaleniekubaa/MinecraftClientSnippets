How to use?
- Just add this button to the buttonList in your Gui
 ```java
 @Override
 public void initGui(){
    System.out.println("Created Gui!");
    this.buttonList.add(new ImageButton(0, 10,10, 20, new ResourceLocation("client/icons/uricon.png")));
    
 }
 ```
